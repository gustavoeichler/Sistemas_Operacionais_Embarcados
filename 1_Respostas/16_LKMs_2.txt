1) O que é memória virtual?
	R: É a utilização do espaço disponível no HD para aumentar a quantidade de memória RAM disponível para rodar novos processos, aumentando a capacidade do sistema.
 
2) Qual é a diferença entre o user space e o kernel space?
	R: A memória RAM é dividida em 2 partes, o user spcae  e o kernel space. O user space é o conjunto de locais disponíveis para que os processos de usuários rodem normalmente. O kernel space é um espaço dedicado ao sistema para rodar o kernel, quando ocorre uma chamada de função do sistema por um processo, uma interrupção é enviada ao kernel para rodar a chada de sistema.

5)Qual é a diferença entre Linux e GNU/Linux?
	R: Linux é o sistema operacional rodando um kernel clone do UNIX. GNU/Linux é um sistema operacional com o kernel linux modificado em relação ao UNIX original da década de 70 e 80.

6)O que é um Linux Kernel Module (LKM)? Quais são suas vantagens?
	R: É um arquivo que contém um exemplo de kernel de um sistema operacional. Sem módulos do kernel carregáveis, um sistema operacional teria que incluir toda a funcionalidade antecipada possível já compilada diretamente no kernel base. Grande parte dessa funcionalidade residiria na memória sem ser usada, desperdiçando memória e exigiria que os usuários reconstruíssem e reinicializassem o kernel base toda vez que exigissem nova funcionalidade. A maioria dos sistemas operacionais que suportam módulos carregáveis ??do kernel incluirá módulos para suportar a funcionalidade mais desejada.
